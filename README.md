## 学习基础

如果你想开心的编写 Solid 程序，那么需要了解关于 Solid 规范的基础概念和 Linked Data 的相关基础知识。在 Solid 程序中，数据模型使用 Linked Data（相当于中心化应用的 MySQL），它驱动了 Solid 的内核。
